# 🎲 Dice Roll Simulator

An interactive and visually polished **Dice Roll Simulator** built using **HTML, CSS, and JavaScript**.  
This project demonstrates DOM manipulation, CSS animations, and clean UI design with custom dice dots (pips).



## ✨ Features

- 🎯 Random dice roll logic (1–6)
- 🎲 Custom dice UI with **large, clear dots (pips)**  
- 💫 Smooth roll animation for better user experience
- 🧾 Roll history tracking (updates after every roll)
- 📱 Responsive & modern UI design
- 🧠 Clean and readable JavaScript logic

----

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling, layout & animations
- **JavaScript (ES6)** – Logic & DOM manipulation

## 🚀 How It Works

1. User clicks on **Roll Dice** button  
2. Dice plays a smooth roll animation  
3. A random number between **1 and 6** is generated  
4. Dice face updates using **custom CSS dots**
5. Roll result is added to the **history list**
