## âš™ï¸ How It Works

1. The quiz starts with a timer for each question.
2. User selects an option and moves to the next question.
3. If the timer ends, the quiz automatically proceeds.
4. The score is calculated based on correct answers.
5. After the last question, the user is redirected to the result page.
6. The result page displays:
   - Total questions
   - Correct answers
   - Wrong answers
   - Final score
   - PASS / FAIL status
