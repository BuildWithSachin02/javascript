# 🎂 Age Calculator

A simple and beginner-friendly **Age Calculator** built using **HTML, CSS, and JavaScript**.  
This project calculates a user’s age in **years, months, days, hours, minutes, and seconds** based on their date of birth.



## ✨ Features

- 📅 User selects date of birth
- ⏳ Calculates exact age in:
  - Years
- 🧠 Beginner-friendly logic
- 🎨 Clean and modern UI
- 🚫 Input validation (date required))




## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & layout
- **JavaScript (Vanilla JS)** – Logic & calculations

## 🚀 How It Works

1. User selects their **date of birth**
2. On button click, JavaScript:
   - Gets today’s date
   - Converts both dates into Date objects
   - Calculates the time difference
3. The difference is converted into:
   - Years
   - Months
   - Days
   - Hours
   - Minutes
   - Seconds
4. Result is displayed on the screen instantly

## ⚠️ Note

This project uses **approximate values**:
- 1 year = 365 days
- 1 month = 30 days  

This approach is **perfect for learning and practice**.  
Exact calendar-based calculation can be added later as an advanced enhancement.
