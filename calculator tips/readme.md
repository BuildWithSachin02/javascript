# 💰 Tip Calculator

A simple and clean **Tip Calculator** built using **HTML, CSS, and JavaScript**.  
This project helps calculate the total bill amount by adding a tip percentage entered by the user.



## 🚀 Features

- Enter bill amount
- Enter tip percentage
- Calculates tip based on percentage
- Displays total payable amount
- Clean and responsive UI
- Beginner-friendly logic

----

## 🛠️ Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling and layout
- **JavaScript (Vanilla JS)** – Logic and calculation

---

## 🧠 Logic Explanation

1. User enters:
   - Bill amount
   - Tip percentage
2. JavaScript converts input values from **string to number**
3. Tip is calculated using the formula:
Tip = (Bill Amount × Tip Percentage) / 100

4. Total amount is calculated as:


Total = Bill Amount + Tip

5. Result is displayed on the screen
