# 🎯 Number Guessing Game

A simple and logic-focused **Number Guessing Game** built using **HTML, CSS, and JavaScript**.  
This project helps strengthen conditional logic, random number generation, and user input handling.



## ✨ Features

- 🔢 Random number generation (1–20)
- 🧠 Logic using `if / else` conditions
- ⌨️ User input validation
- 📢 Feedback system (Too High / Too Low / Correct)
- 🎨 Clean and simple UI

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & layout
- **JavaScript** – Game logic

---

## 🚀 How the Game Works

1. The system generates a random number between **1 and 20**
2. The user enters a guess
3. The app compares the guess with the random number
4. Feedback is displayed:
   - 📈 Too High
   - 📉 Too Low
   - 🎉 Correct Guess
